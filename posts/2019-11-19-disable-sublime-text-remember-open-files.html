取消 Sublime Text 打开时记住上次打开的文档

经常会使用 Sublime Text 3 来编辑文档，比如一些 Markdown 文档或者是一些源代码文件，但是每次打开的时候都会自动打开上次打开过的文档。这时 Sublime Text 就会打开多个窗口，这比较影响编辑文档时候的专注力和效率，对于多打开的文档，笔者比较习惯先关闭再来编辑需要编辑的文档。

一、首先打开 Sublime Text，打开菜单栏中的 Preferences -> Settings

二、在这段代码的中间增加以下两行代码

"hot_exit": false,
"remember_open_files": false,

三、最后附上一张笔者的 Sublime Text 3的设置完成代码

{
    "color_scheme": "Packages/Color Scheme - Default/Monokai.sublime-color-scheme",
    "font_face": "Monaco",
    "font_size": 16,
    "hot_exit": false,
    "remember_open_files": false,
    "tab_size": 4,
    "theme": "Default.sublime-theme",
    "translate_tabs_to_spaces": true,
    "update_check": false
}
