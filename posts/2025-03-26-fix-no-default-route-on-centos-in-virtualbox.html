【已解决】CentOS系列系统在VirtualBox中Host-Only网络模式下没有默认路由

  1. 一、现象
  2. 二、解决方法
  3. 三、结果

一、现象

执行 $ ip route 命令，输出中没有默认路由。

[root@localhost ~]# ip route
192.168.56.0/24 dev enp0s3 proto kernel scope link src 192.168.56.102 metric 100

二、解决方法

编辑以下文件并重启系统：

/etc/sysconfig/network-scripts/ifcfg-enp0s3

参数配置如下（有则修改，没有则添加）：

BOOTPROTO=none
IPADDR=192.168.56.101
PREFIX=24
GATEWAY=192.168.56.1

三、结果

[root@localhost ~]# ip route
default via 192.168.56.1 dev enp0s3 proto static metric 100 
192.168.56.0/24 dev enp0s3 proto kernel scope link src 192.168.56.101 metric 100
