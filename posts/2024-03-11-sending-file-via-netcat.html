Sending File via Netcat

2024-03-11 / Linux Network

For example, your want send a file named hello.zip from client to server which ip and port are 192.168.56.102 and 8080

On your server:

$ nc -l -p 8080 -q 1 > hello.zip < /dev/null

On your client:

$ cat hello.zip | nc 192.168.56.102 8080
